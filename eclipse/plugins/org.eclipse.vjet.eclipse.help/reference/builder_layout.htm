VJET Builder

The VJET builder also known as Script builder builds VJET programs using a compiler that implements the VJET Language Specification. The VJET builder can build programs incrementally.

You can reach the Builder Page through the Project window (Project > Builders) from the context menu on a created project or the File menu of the workbench.

Builder options
Option Description
New.. This command allows you have a new builder for the VJET project.
edit Edit the selected builder.
The default builder: Script Builder cannot be edited
